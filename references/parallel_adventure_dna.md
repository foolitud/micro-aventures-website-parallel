# Parallel Adventure DNA - Reference Document

This document serves as the core reference for the Parallel Adventure ecosystem. It should be used to inform analyses, projects, and content creation **only when explicitly requested by the user**.

---

## 1. Parallel Adventure: L'Expérience Joueur Grandeur Nature
**The Player Experience & Catalog.**
- **Concept:** Players become protagonists of real-life adventures (investigation, fiction, treasure hunts).
- **Gameplay:** Interaction with NPCs via messages/calls, internet searches, decoding, and real-life location visits or physical mail.
- **Impact:** Decisions have real consequences, affecting character paths and overall success.

## 2. Parallel Builder: L'Atelier de Création Interactif
**The Public Creation Tool.**
- **Objective:** Democratize the creation of immersive adventures for everyone.
- **Ambition:** Engage the community in co-creating the universe and scaling the number of stories.

## 3. Parallel Studio: L'Incubateur de Talents et de Scénarios
**Incubator & Production Support.**
- **Function:** Monthly selection of two screenwriters to help them produce professional-grade adventures.
- **Support:** End-to-end guidance (media creation, game mechanics, gamification, testing, and launch).

## 4. Parallel Engine: Le Cœur Technologique de la Réalité Alternée
**The Technical Infrastructure.**
- **Real-Life First:** Specifically built for physical immersion and ARG, not traditional screens.
- **Native "No-App" Experience:** Zero-friction interaction via real-world channels (WhatsApp, SMS, calls).
- **Deep Game Logic:** Advanced state machine managing complex logic, variables, and real-world consequences.
- **Vertically Integrated:** Sovereignty over both creation (Builder) and runtime (Execution).
